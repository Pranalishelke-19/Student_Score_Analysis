# 🎓 Student Score Analysis (Python Project)

## 📌 Project Overview
This project analyzes student performance using Python. It focuses on understanding how different factors affect scores in Math, Reading, and Writing.

---

## 📊 Dataset Features
The dataset includes:

- Gender  
- EthnicGroup  
- ParentEduc  
- LunchType  
- TestPrep  
- ParentMaritalStatus  
- PracticeSport  
- IsFirstChild  
- NrSiblings  
- TransportMeans  
- WklyStudyHours  
- MathScore  
- ReadingScore  
- WritingScore  

---

## 🎯 Objectives
- Analyze student performance  
- Find factors affecting scores  
- Compare performance based on study hours, gender, etc.

---

## 🛠️ Tools & Libraries Used
- Python  
- Pandas  
- NumPy  
- Matplotlib / Seaborn  

---

## 📈 Analysis Performed
- Data cleaning and preprocessing  
- Exploratory Data Analysis (EDA)  
- Visualization of scores  
- Comparison of different features  

---

## 🔍 Key Insights
- Students with more study hours scored higher  
- Test preparation improved performance  
- Parental education had an impact on scores
